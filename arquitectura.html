Reglas importantes al modificar este proyecto No agregar frameworks ni build tools — todo es vanilla JS con <script src>. No modificar packing.js sin análisis previo (BFD + heightmap con priority zones, interlocking y posicionamiento manual). No separar más los archivos JS. Al agregar globals, verificar que no rompan archivos que se cargan después. La app usa fetch() para cargar html/ip.html y html/cl.html — requiere servidor HTTP (no funciona con file://).
